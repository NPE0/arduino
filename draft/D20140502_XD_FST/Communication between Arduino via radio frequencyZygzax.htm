Communication between Arduino via radio frequency

36 Replies

Hello, the other day I bought on eBay this RF module for Arduino:

  • FS1000A



This module includes a receiver and a transmitter operating in the frequency 315MHz, the connections with Arduino are simple:

In the case of the transmitter (XY-FST):

  • ATAD (Data) —> Digital pin 3 (You can change to another pin in the program code).
  • VCC —> +5V
  • GND —> GND

For the receiver (XY-RF-5V):

  • GND —> GND
  • DATA —> Digital pin 3 (You can change to another pin in the program code).
  • DATA —> We can leave this pin unconnected, we will use only one data pin (We can use either of them).
  • VCC —> +5V

Before loading the code you need to download and install the library VirtualWire, The direct download link is the following:

http://www.open.com.au/mikem/arduino/VirtualWire/VirtualWire-1.14.zip

The following code allows you to send and receive a message between two Arduino boards:

Code for the transmitter:

Data hosted with ♥ by Pastebin.com - Download Raw - See Original
  1. //Test and text transmission with RF module for Arduino http://zygzax.com
  2. //TRANSMITTER
  3. #include <VirtualWire.h>
  4. void setup()
  5. {
  6.     vw_set_ptt_inverted(true);  // Required by the RF module
  7.     vw_setup(2000);            // bps connection speed
  8.     vw_set_tx_pin(3);         // Arduino pin to connect the receiver data pin
  9. }
  10.  
  11. void loop()
  12. {
  13.    //Message to send:
  14.    const char *msg = "HELLO WORLD";
  15.    vw_send((uint8_t *)msg, strlen(msg));
  16.    vw_wait_tx();        // We wait to finish sending the message
  17.    delay(200);         // We wait to send the message again                
  18. }

Code for the receiver:

Data hosted with ♥ by Pastebin.com - Download Raw - See Original
  1. //Test and text transmission with RF module for Arduino http://zygzax.com
  2. //RECEIVER
  3. #include <VirtualWire.h>
  4.  
  5. void setup()
  6. {
  7.     Serial.begin(9600);          // Configure the serial connection to the computer
  8.     vw_set_ptt_inverted(true);  // Required by the RF module
  9.     vw_setup(2000);            // bps connection speed
  10.     vw_set_rx_pin(3);         // Arduino pin to connect the receiver data pin
  11.     vw_rx_start();           // Start the receiver
  12. }
  13.  
  14. void loop()
  15. {
  16.   uint8_t buf[VW_MAX_MESSAGE_LEN];
  17.   uint8_t buflen = VW_MAX_MESSAGE_LEN;
  18.   if (vw_get_message(buf, &buflen))      // We check if we have received data
  19.   {
  20.     int i;
  21.     // Message with proper check    
  22.     for (i = 0; i < buflen; i++)
  23.     {
  24.       Serial.write(buf[i]);  // The received data is stored in the buffer
  25.                             // and sent through the serial port to the computer
  26.     }
  27.     Serial.println();
  28.   }
  29. }

Once loaded and running the programs, connecting the serial monitor of the Arduino application with the receiver board to 9600, we should see the message that we are sending from the transmitter board.
